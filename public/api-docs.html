POST gen.aditor.ai/api/animate-alpha internal

Takes an image URL and a text prompt. Generates an animated video via Kling v3 (fal.ai), strips the background frame-by-frame using rembg, and outputs a looping WebM VP9 file with full alpha channel (transparent background).

Built for use as canvas elements in AFFiNE (plan.aditor.ai). Paste the returned embed_html into any AFFiNE HTML block — it will autoplay, loop, and have no background.

Request Body

FieldTypeNotes
image_urlstringrequired  ·  publicly accessible URL (must be reachable by fal.ai CDN — no Wikipedia/blocked hosts)
promptstringrequired  ·  animation instruction e.g. "spin this cube slowly clockwise"

Example Request

{ "image_url": "https://cdn.gen.aditor.ai/assets/product.png", "prompt": "spin this object slowly, smooth 360 rotation" }

Example Response

{ "success": true, "job_id": "a3f2c1d9-e7b3-4a5f-8c2d-1e6f9a0b4c7e", "webm_url": "https://gen.aditor.ai/webm/a3f2c1d9-e7b3-4a5f-8c2d-1e6f9a0b4c7e.webm", "embed_html": "<video autoplay loop muted playsinline style='background:transparent'><source src='https://gen.aditor.ai/webm/a3f2c1d9....webm' type='video/webm'></video>", "note": "Paste embed_html into an AFFiNE HTML block for a transparent looping animation." }

AFFiNE Usage

// AFFiNE canvas → Insert → HTML Block → paste embed_html // Autoplays, loops, transparent background. // Resize freely. No additional requests after first load.
Pipeline: Kling v3 standard (fal.ai) → ffmpeg frame extraction → rembg background removal → ffmpeg VP9/yuva420p WebM
Output served from: gen.aditor.ai/webm/
Avg processing time: ~90–120s for a 5s clip at 24fps
